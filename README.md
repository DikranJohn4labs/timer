# Countdown Timer (Python)

## Description

This project is a simple countdown timer written in Python.
The user enters a time in the format **hours:minutes:seconds (h:m:s)** and the program counts down every second in the terminal until it reaches **00:00:00**.

## Features

* Takes time input from the user
* Converts the time into seconds
* Displays the countdown in the terminal
* Updates every **1 second**

## Requirements Used

This project includes the following programming concepts:

* **Variables** – storing time values
* **Conditions** – controlling the countdown loop
* **Functions** – using a function to run the timer
* **Loops** – `while` loop for countdown
* **Arrays (Lists)** – splitting the input time into parts
* **Strings** – formatting and displaying time
* **Library** – `time` module for 1-second delay

## How to Run

1. Make sure Python is installed.
2. Download or clone the repository.
3. Run the program:

```bash
python timer.py
```

4. Enter a time in the format:

```
h:m:s
```

Example:

```
0:0:10
```
